Desktop Search Engine
Software Version 1.0

Desktop Search Engine is based on Whoosh. It is implemented by python and can retreive local PC information in the same time.
It provides TF-IDF for scoring files, stemming for indexing files, MajorClust for clustering results.

It is developed under Ubuntn 16.04.
Here is what you need: python 2.7, wxpython 2.8, whoosh, open-docx, odt2txt, catdoc, pdfminer.

To run it, just type:

python final.py
